Add a new list to the web page

Add elements to the DOM

Let's practice what we learned by adding new content to the web page. We want to finish the web page by adding names of awesome women.

Hey Slacker!

Remember, we're here to help. Join the KCWiT #codingandcocktails Slack Channel: kcwit.slack.com

  1. Using Atom, declare a new array to hold the names of awesome women after the brainSkills array in scripts.js. You can use whatever variable name you want.

    Need a little help? Expand this section for guidance. Type const ladyDevs = [];.

  2. Populate the array with names of women you met at Coding & Cocktails. If you can't remember any names, here's some awesome lady devs to use:

    • Grace Hopper
    • Evelyn Boyd Granville
    • Ada Lovelace
    • Barbara Liskov
    • Margaret Hamilton
    • Sister Mary Kenneth Keller

  3. Find the declaration for the function addDevSkill at the top of scripts.js. After the addDevSkill function closing curly brace, declare a new function to use for writing the names of the lady devs to the DOM. Remember, function names are verbs and parameters are nouns.

    Need a little help? Expand this section for guidance. Create a new function and name it after the action (the verb). In this example, the action is `addLadyDev`. You can use Function expression or Function declaration. Your function should look something like this: 
    
const addLadyDev = function (ladyDev) {
};

  4. Between the opening and closing curly braces of your new function, type the following code to create a new list element, set the text, and add it to the DOM:

    let listItem = document.createElement('li');
listItem.innerHTML = <your function's parameter for lady devs>;
document.getElementById("fellow-ladydevs").appendChild(listItem);

    This is a sneak peek at a concept we'll cover in more detail next session-- DOM manipulation. Here we are creating a list dynamically in JavaScript and adding it to the webpage. Now you're extra prepared for next session!

  5. In the document.addEventListener("DOMContentLoaded", ...) function at the bottom of scripts.js, iterate through the new array and call your newly created function.

    Need a little help? Expand this section for guidance. Take a look at how you iterated over brainSkills and called the addDevSkill function. You'll do the same for ladyDevs array and calling addLadyDev function.

  6. Save your file and start atom-live-server. Do you see the names of fellow LadyDevs? Upload a picture of your web page to Slack and show off your work!
